# Simple PHP Form Project

## Features
- Secure login/logout
- Public form to collect submissions
- Admin panel to view submitted data

## Tech
- PHP 7+
- MySQL
- PDO

## Setup
1. Import `sql/schema.sql` into MySQL.
2. Configure DB credentials in `config/db.php`.
3. Manually create an admin user using a temporary script or SQL.
